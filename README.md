# amour-recipe-platform
Full-stack recipe CRUD web application built with Spring Boot, Thymeleaf, and SQL (In Progress)

Tech Stack
- Java
- Thymeleaf
- MySql (H2 currently for testing)
- Gradle for building/managing dependencies

Features (Current and Planned)

- Recipe listings with detailed views
- Full CRUD functionality
- 3-Layer Architecture
- SSR with Thymeleaf
